## ✨ About

Sanjana's Desktop reimagines the traditional developer portfolio as an interactive desktop environment. Instead of scrolling through sections, visitors can open windows, navigate folders, and explore projects just like using a real computer.

The goal was to create a memorable experience while showcasing my work, technical skills, and personality.

- 🖥️ macOS-inspired desktop interface

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/ssanjanasharma/sanjanas-desktop.git
```

Move into the project

```bash
cd sanjanas-desktop
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
